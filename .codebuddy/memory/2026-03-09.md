## V0.4 功能开发 — 年度开发者报告 & Git 实时监听

### 年度开发者报告 (Annual Developer Report)
- **后端**: 新建 `src/devdiary/api/routes/report.py`
  - `GET /api/report?year=YYYY` — 聚合年度所有 commits/diaries，计算月度趋势、活跃天数、项目排行、语言分布、成就系统、亮点（最高产月、最长连续、最忙日、高效时段）
- **前端**: 新建 `frontend/src/views/AnnualReportView.vue`
  - Hero 统计区域 + 4 张 ECharts 图表（月度提交柱状图、活跃天数面积图、项目排行横向柱状图、语言分布环形图）
  - 年份切换器、成就卡片网格、响应式布局
- **CLI**: `devdiary report [--year YYYY]` — 终端显示年度总览表格 + 月度趋势 ASCII 柱状图 + 项目排行榜

### Git 实时监听 (Watcher Service)
- **后端核心**: 新建 `src/devdiary/watcher.py`
  - `WatchConfig` 数据类 — YAML 配置持久化 (`.devdiary/watch.yaml`)
  - `WatcherState` — 运行时状态追踪
  - `run_watcher()` — 异步主循环，支持 daily/weekly/on_push 三种调度模式
  - `_check_and_generate()` — 扫描仓库新提交，AI 生成日记，去重检查
  - `start_watcher_background()` / `stop_watcher()` — 后台任务管理
- **后端 API**: 新建 `src/devdiary/api/routes/watcher.py`
  - `GET /api/watcher/status` / `POST /api/watcher/start` / `POST /api/watcher/stop`
  - `GET /api/watcher/config` / `PUT /api/watcher/config`
- **前端**: 新建 `frontend/src/views/WatcherView.vue`
  - 服务状态卡片（运行/停止、上次检查/生成、下次执行、已生成计数）
  - 启动/停止按钮 + 30秒轮询刷新
  - 配置表单（调度模式、时间、星期、通知设置）
- **CLI**: `devdiary watch [--start/--stop/--status]` — 终端管理监听服务
